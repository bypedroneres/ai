# WLLPR

Procedural desktop wallpapers for macOS and Windows.

![WLLPR](screenshot.png)

## Features

- Procedural wallpaper generation powered by a canvas-based pattern engine
- Multiple wallpaper pattern collections arranged in carousel-style sections
- Featured hero wallpaper section for quick discovery
- Dark and light theme support
- Custom wallpaper creation flow
- One-click wallpaper setting with native macOS and Windows integrations
- One-click export to your Downloads folder
- Cross-platform desktop app built with Tauri v2

## Download

Get the latest Windows build from [GitHub Releases](https://github.com/bypedroneres/ai/releases).

## Build from Source

### Prerequisites

- Rust
- Node.js

### Commands

```bash
npm install
npx tauri build
```

## Tech Stack

- Tauri v2
- Rust
- Vanilla HTML, CSS, and JavaScript
- Canvas API
